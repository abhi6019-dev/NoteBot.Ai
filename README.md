## ✨ What Notebot does

Notebot AI is a **ChatGPT-style study assistant** designed for students. It helps users turn:

- 📸 screenshots and lecture photos
- 📝 rough notes
- 📂 uploaded images
- 🧠 messy ideas

into:

- 📚 structured revision notes
- 🧠 flashcards
- ❓ quizzes
- 🧑‍🏫 explanations
- 📄 exportable PDFs

All of it happens in a **clean, premium, streaming AI interface** with memory and history.

## 🔐 Environment variables

Backend environment variables:

```bash
GROQ_API_KEY=your_groq_key
GROQ_MODEL=llama-3.1-8b-instant
SUPABASE_URL=your_supabase_url
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
CORS_ORIGIN=https://abhi6019-dev.github.io
```
